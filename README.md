# ZAINFUL Client 🚀

> **The Ultimate Minecraft Pocket Edition Hack & Visual Enhancer**

Welcome to **ZAINFUL**, a high-performance client designed for Bedrock Edition players who want more control, speed, and style. Built with C++ for maximum efficiency and injected seamlessly into the game engine.

## ✨ Features
- 🚀 **Fly Mode**: Smooth vertical movement with custom speeds.
- ⚡ **Speed Hack**: Move faster than vanilla limits (configurable).
- 💥 **KillAura**: Auto-attack nearby enemies within a set radius.
- 🎨 **Visuals**: Custom shaders and GUI overlays.
- ⚙️ **Config System**: Save and load your favorite settings instantly.

## 🛠 Installation
1. Download the latest release from the [Releases](link) tab.
2. Extract the files into your Minecraft PE directory.
3. Launch the game with the included launcher (or inject via BlockLauncher).
